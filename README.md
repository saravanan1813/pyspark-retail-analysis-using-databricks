# PySpark Retail Analysis using Databricks

##  Project Overview
This project performs end-to-end retail data analysis using PySpark in Databricks.  
It includes data ingestion, schema definition, transformation, and analysis.

---

##  Features
- Defined structured schema using StructType
- Processed retail datasets using PySpark
- Performed data transformations and analysis
- Generated insights from sales data

---

##  Architecture
Medallion Architecture:
- Bronze Layer: Raw data ingestion
- Silver Layer: Cleaned and transformed data
- Gold Layer: Business insights

---

## Tech Stack
- PySpark
- Databricks
- Python
